## Why FlakeTriage?

- **Zero-config** — works immediately with safe defaults after install
- **Evidence** — every decision posted as a Check Run with full audit trail
- **Impact visibility** — estimated developer time saved per suspected flake
- **Budget control** — configurable daily rerun limits per repository
- **Privacy** — operates on workflow metadata only, never reads source code or logs
