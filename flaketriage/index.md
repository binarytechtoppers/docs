## Key Features

- **Automatic rerun** of failed CI jobs with single-rerun idempotency
- **Check Run evidence** on every commit — full decision audit trail
- **Impact visibility** — estimated developer time saved, weekly digest
- **Policy engine** — built-in deny patterns for deploy/release workflows
- **Budget control** — configurable daily limits per repo, capped by plan tier
- **Manual retry** — `/flaketriage retry <run_id>` for on-demand reruns
- **Zero-config** — safe defaults, no setup file required
